Available component type names
--------------------------------------------
You can combine many component types. Example: componentTypes = "EntityClasses, HighLevelMaterialClasses, GUIControlClasses".

RenderingSystem - rendering system implementation
PhysicsSystem - physics system implementation
SoundSystem - sound system implementation

HighLevelMaterialClasses - high level material class definitions
CompositorInstanceClasses - full-screen effect class definitions
GUIControlClasses - GUI system class definitions
