Official NeoAxis 3D Engine Exporter for Autodesk Maya

===========================================

Features:

	- Mesh export
	- Scene export support
	- Material export
		- Supported properties of Maya materials for maps:
			- Color
			- Specular color
			- Ambient color
			- Bump Mapping
		- Supported properties of Maya materials for colors:
			- Color
			- Specular color
			- Ambient color
		- Supported other Maya material's parameters
			- Eccentricity
			- Transparency
	- Texture export (Automatic converting to JPG, PNG, TGA, DDS-DXT1, DDS-DXT5, DDS-3Dc)
	- Vertex colors
	- Normals
	- Skeleton animation (supports the Skin)
	- Morph animation
	- Pose animation (supports the Blend shape)
	- Multiple texture coordinate channels
	- Edge list generation

===========================================

Installation:

	- Run NeoAxisEngineExporterInstaller_Maya.exe, select Maya version and press "Install".
	- Run Maya.
	- Go to main menu, select "NeoAxis Engine Export".
