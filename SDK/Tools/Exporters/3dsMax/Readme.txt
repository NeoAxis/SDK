Official NeoAxis 3D Engine Exporter for 3D Studio Max

===========================================

Features:

	- Mesh export
	- Scene export support
	- Material export (high level materials)
		- Supported properties of 3dsMax materials for maps:
			- Diffuse Color
			- Specular Color
			- Self-Illimination
			- Bump
		- Supported properties of 3dsMax materials for colors:
			- Diffuse Color
			- Specular Color
			- Self-Illimination
		- Supported other 3dsMax material's parameters
			- Glossiness
			- Opacity
			- Culling
	- Texture export (Automatic converting to JPG, PNG, TGA, DDS-DXT1, DDS-DXT5, DDS-3Dc)
	- Skeleton animation (Skin modifier)
	- Morph animation
	- Pose animation (Morpher modifier)
	- 3dsMax Multi-material support
	- Vertex colors
	- Normals and tangent vectors
	- Multiple texture coordinate channels
	- Edge list generation

===========================================

Installation:

	You can look tutorial here:
		http://www.neoaxisgroup.com/wiki/Documentation/Tutorials/Export_models_from_Autodesk_3ds_Max
	Video tutorial:
		http://www.neoaxisgroup.com/wiki/Documentation/Video_Tutorials

	=====

	- Run NeoAxisEngineExporterInstaller_3dsMax.exe, select 3ds Max version and press "Install".
	- Run 3ds Max.
	- Go to "Command panel", select "Utilities" page.
	- Click "More..." and select "NeoAxis Engine Exporter" in the list.
