How to make font:
- Open Bitmap Font Generator.
- Options -> Font Settings -> Check the file format as XML, not a Text.
- Options -> Font Settings -> Check the file format as 32-bit image format, not a 8-bit.
- Look at the right side of the Bitmap Font Generator,select those checks on.
  Such as Basic Latin + Sypplement etc....
- Save bitmap font as... And then you will get some bitmaps.
- Convert the font file into fontDefinition files by means BitmapFontGeneratorToNeoAxisConverter. 
  (Warning: Be sure the fnt file's name is English or numbers,DO NOT USE Russian or anything else.)
  And copy these fontDefinition files to the \Data\Fonts\ folder.
- Now you can use new font.
